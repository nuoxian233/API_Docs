## 接口描述

> 获取指定网站的网站图标

## 接口地址

请求方式：`GET`

返回格式：`ICO`

```API
https://api.nxvav.cn/api/wzico/
```

## 请求示例

[https://api.nxvav.cn/api/wzico/?url=nxvav.cn](https://api.nxvav.cn/api/wzico/?url=nxvav.cn)

#### 使用场景

<!-- tabs:start -->

```html
<img src="https://api.nxvav.cn/api/wzico/?url=nxvav.cn" />
```
